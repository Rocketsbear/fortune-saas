# 命理融合咨询 Pro - Open Graph 图片

## 说明

在 `frontend/public/` 目录下添加 `og-image.png` (1200x630 像素)

### 推荐生成方式

1. **使用 Nano Banana Pro** (你的技能):
   ```
   生成一张高端深色风格的命理咨询分享卡片，包含：
   - 标题：命理融合咨询 Pro
   - 紫色/金色点缀
   - 玄学/占卜元素
   - 尺寸：1200x630
   ```

2. **使用在线工具**:
   - https://og-image.vercel.app
   - https://www.cardd.io

3. **手动设计**:
   - 使用 Figma/Canva 设计
   - 包含：Logo、标题、描述、背景图

### 需要的图片文件

```
frontend/public/
├── og-image.png        # 1200x630 Open Graph 图片
├── twitter-image.png   # Twitter 分享图 (可选)
└── favicon.ico         # 网站图标
```
